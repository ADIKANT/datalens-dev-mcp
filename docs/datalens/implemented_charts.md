# Implemented Charts

This catalog is generated by `update_implemented_charts_catalog` during
DataLens bundle generation. Project-specific runs write implemented chart
details into:

- `docs/datalens/implemented_charts.md`
- `requirements/charts.md`
- `requirements/metrics.md`
- `requirements/object_relations.md`

No implemented charts are recorded in the repository template workspace yet.

## Required Entry Fields

Each generated entry includes chart ID, dashboard/page/tab, chart family,
implementation path, template used, dataset/source, dimensions, measures,
calculated fields, filters, selectors, object relations, native title/hint
metadata, style/theme notes, known limitations, and last update timestamp.

Native title/hint metadata is recorded as `title`, `hint`, `hideTitle`, and
`enableHint`. This lets a reviewer verify that Advanced Editor chart bodies did
not duplicate dashboard-level titles or hints.

## Historical Conversion Notes

Removed chart variants are not listed as current implementations. If a removed
chart is found in a legacy dashboard, record it here as historical conversion
evidence only.
