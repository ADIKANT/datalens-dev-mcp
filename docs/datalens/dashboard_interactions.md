# Dashboard Interactions

Source trace: dashboard relation schema, official selector/parameter/link docs, Dashboard System UX guidance and demo workbook harvest plan.

- Relation types: selector filters chart, selector controls parameter, chart click filters chart, chart click navigates to dashboard, overview drilldown to object page, tab shared filters, embedded links/actions.
- Generated dashboard specs must declare relation artifacts before payload planning.
- Validation checks that target widgets/charts exist and that selector source fields are compatible with dataset fields.
- Selector rows use left labels and keep multi-control width totals at or below 94%.
