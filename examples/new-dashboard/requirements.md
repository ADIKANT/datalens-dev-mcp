# Synthetic New Dashboard Requirements

Create a dashboard for a public sample subscription service.

- Audience: operations and product analysts.
- Main questions: active subscriptions, weekly trend, top plan contribution, and account detail lookup.
- Data: `examples/new-dashboard/sample.csv`.
- Routes: KPI and trend use `editor_advanced`; detail lookup uses `editor_table`; notes use `editor_markdown`; segment filter uses `editor_js_control`.
