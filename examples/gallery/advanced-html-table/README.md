`table-custom-html/` is the only rare custom Advanced table example surface.

Default generated tables use `../table-widget/` on `editor_table` / `table_node`. Use this Advanced example when the requested widget needs grouped headers, sticky columns, or layout behavior that `table_node` cannot express.

The route is still `editor_advanced` with the standard Advanced file set:

- `meta.json`
- `params.js`
- `sources.js`
- `prepare.js`

Do not use this folder as the default table route and do not add `config.js`.
