# Requirements templates

Copy these files into a project-local `requirements/` directory and fill them with the dashboard brief, data model, metrics, fields, selectors, relations, and user decisions.

Use `dl_validate_project` to check the workspace, `dl_build_payload_plan` to compile validated materials, and `dl_create_safe_apply_plan` to choose plan-only, save-only, or save-and-publish from the user request.
